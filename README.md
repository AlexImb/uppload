# Uppload

Uopload is a better JavaScript file uploader.

## Usage

Import Uppload to your project:

```
yarn add uppload
```

## Development

### Roadmap
- Modal
- File preview
- Image cropping
- Upload function
- Import file/image (Facebook, Dropbox, etc.)

### Installation

```
yarn install
nvm use 8.11.1
```

### Production

```
yarn build
```
