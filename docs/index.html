Copy and paste the following code to your head:

<script src="https://unpkg.com/uppload/dist/uppload.no-polyfills.min.js"></script>

Add the vanilla JS Uppload plugin:

yarn add uppload

For Vue.js:

yarn add uppload-vue

For React:

yarn add uppload-react
